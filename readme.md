# Relativity Simulator

An interactive platform to explore Special and General Relativity through
visualizations, mathematics, and physical intuition.

## Features
- Special Relativity: Time dilation, Lorentz transformations, Minkowski spacetime
- General Relativity: Black holes, spacetime curvature, cosmology
- Interactive simulations and animations

## Built With
- Python
- Streamlit
- NumPy
- Matplotlib
- Plotly


## Run Locally
pip install -r requirements.txt  
streamlit run app.py
